# Second Project

## 📌 Description
This is a simple C project where the user has to guess a number generated by the computer.

## ⚙️ Features
- Random number generation
- User input handling
- Hints (higher/lower)

## 🛠️ Tech Used
- C Programming Language

## ▶️ How to Run
1. Compile the code:
   gcc guess_number.c -o guess
2. Run the program:
   ./guess

## 📂 File Structure
- guess_number.c → main source code

- Add difficulty levels
